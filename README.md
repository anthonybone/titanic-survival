# Titanic Survival Prediction

This is a beginner data science project that predicts passenger survival using the Titanic dataset.

## Tools Used
- Python
- pandas
- seaborn / matplotlib
- scikit-learn

## Features
- Data cleaning and imputation
- Logistic Regression classification
- Evaluation with accuracy, confusion matrix, and classification report
- Visualization of feature relationships

## Dataset
[Titanic CSV from DataScienceDojo](https://raw.githubusercontent.com/datasciencedojo/datasets/master/titanic.csv)
